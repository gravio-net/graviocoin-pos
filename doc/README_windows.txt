Graviocoin Core

Intro
-----
Graviocoin is an open source, decentralized privacy platform
built for global person to person eCommerce.

For more information, as well as an immediately useable, binary version of
the Graviocoin Core software, see https://graviocoin.io.


Setup
-----
Unpack the files into a directory and run graviocoin-qt.exe.

Graviocoin Core is the original Graviocoin client and it builds the backbone of the network.
However, it downloads and stores the entire history of Graviocoin transactions;
depending on the speed of your computer and network connection, the synchronization
process can take anywhere from a few hours to a day or more.

See the graviocoin wiki at:
  https://graviocoin.wiki/
for more help and information.
